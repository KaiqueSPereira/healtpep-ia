# Gerenciamento de Exames

Esta seção da documentação detalha como gerenciar seus exames laboratoriais e de imagem no sistema.

## Visualizando Exames

Na tela de exames, você tem duas opções de visualização:

1.  **Gráfico de Evolução:** Acompanhe a evolução dos resultados de seus exames ao longo do tempo. Você pode filtrar os tipos de exames (sangue, urina, etc.) para uma análise mais focada.
2.  **Lista de Exames:** Veja uma lista completa de todos os exames cadastrados, organizados em cards para fácil visualização.

## Cadastrando um Novo Exame

1.  Clique no botão **"Novo Exame"** na tela inicial.
2.  Você pode vincular o exame a uma **consulta existente** ou selecionar manualmente o **médico** e a **unidade de saúde/laboratório** responsável.
3.  Preencha os detalhes do exame, como data, motivo e tipo.
4.  **Anexar Arquivos:**
    *   Para **exames de imagem**, anexe o arquivo correspondente (PDF, imagem, etc.).
    *   Para **exames de sangue ou urina**, uma tabela será exibida para preenchimento dos resultados.
5.  **Preenchimento de Dados:**
    *   **Manual:** Insira os valores dos exames manualmente.
    *   **Leitura com IA:** Utilize a função de inteligência artificial para extrair os dados automaticamente de um arquivo de resultado. Após a extração, você poderá conferir e ajustar os dados antes de salvar.

## Editando e Detalhando Exames

*   **Editar:** Na tela de listagem de exames, cada card possui um botão para edição, permitindo que você altere os dados do exame a qualquer momento.
*   **Visualizar Detalhes:** Clicando em um card de exame, você será redirecionado para uma tela com todos os detalhes específicos, incluindo resultados, anexos, data, laboratório e anotações.

## Filtrando Exames

Para facilitar a busca, você pode filtrar os exames com base nos seguintes critérios:

*   Médico solicitante
*   Tratamento associado
*   Unidade de saúde/laboratório

Isso permite uma organização e acesso rápido às informações que você precisa.
